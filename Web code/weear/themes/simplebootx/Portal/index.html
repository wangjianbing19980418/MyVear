什么是微耳朵(Vear)

对传统聊天工具的一个优化升级，基于先进的互联网技术、移动互联网技术和浏览器提供的专业获取信息技术，在有网的情况下实现实时交流的“用户、并发”的用户平台

微耳朵(Vear)的研发团队

主要是来自大二的学生经过一个暑假的时间来设计研发,还有许多不足之处,对产品的理解不是很清楚，技术也不是很熟,但我们用心做好app来为社会服务

微耳朵(Vear)的稳定优势

该产品主要为了解决学生，青年收听广播难的问题,而且借用手机的使用聆听移动平台来实现,该产品运用对讲云平台+途聆客户端+途聆对讲配件解决要解决因为移动网络不稳定带来的层出不穷的对讲延迟、卡顿等问题，让对讲用户尽可能保持良好的体验

微耳朵(Vear)的安全策略

提供的稳健的安全策略，包括备份恢复，容错，防治恶意攻击登陆，app防篡改等多项安全管理功能，保证系统安全，可靠，稳定的运行。

微耳朵(Vear)的应用模块化

在原有的APP基础上,提出全新的应用模式并进行扩展,各个模块虽有联系,但模块化的app设计方案使用起来却容易上手

微耳朵(Vear)的免费策略

app遵循app开发协议，免费使用，对商业用户也无任何限制。
